# Changelog

All notable changes to this project will be documented in this file.

The format is based on [Keep a Changelog](https://keepachangelog.com/en/1.0.0/),
and this project adheres to [Semantic Versioning](https://semver.org/spec/v2.0.0.html).

## [1.0.0] - 2025-01-XX

### Added
- Initial release of Analos Launch SDK
- Network configuration helpers for Analos blockchain
- Connection utilities optimized for Analos RPC
- Program deployment command generation
- Program verification utilities
- Network status monitoring
- Balance checking utilities
- Explorer URL generation
- Cost estimation for deployments
- Utility functions (keypair loading, sleep, etc.)

### Features
- `createAnalosConnection()` - Create optimized connections to Analos
- `getDeploymentCommand()` - Generate Solana CLI deployment commands
- `getUpgradeCommand()` - Generate program upgrade commands
- `estimateDeploymentCost()` - Estimate deployment costs
- `verifyProgram()` - Verify program deployment status
- `getNetworkStatus()` - Check Analos network health
- `getBalance()` - Get account balances
- `getExplorerUrl()` - Generate explorer URLs
- `loadKeypair()` - Load keypairs from files

### Documentation
- Comprehensive README with examples
- API reference documentation
- Usage guides and examples
