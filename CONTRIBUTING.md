# Contributing to @analos/launch-sdk

Thank you for your interest in contributing to the Analos Launch SDK!

## Getting Started

1. Fork the repository
2. Clone your fork: `git clone https://github.com/YOUR_USERNAME/analos-launch-sdk.git`
3. Install dependencies: `npm install`
4. Create a branch: `git checkout -b feature/your-feature-name`

## Development

### Building

```bash
npm run build
```

### Testing

```bash
npm test
```

## Code Style

- Use TypeScript
- Follow existing code style
- Add JSDoc comments for public functions
- Keep functions focused and small

## Pull Requests

1. Make sure your code builds: `npm run build`
2. Update documentation if needed
3. Add tests for new features
4. Submit a pull request with a clear description

## Reporting Issues

Please use GitHub Issues to report bugs or request features. Include:
- Description of the issue
- Steps to reproduce
- Expected behavior
- Actual behavior
- Environment details

## Questions?

Feel free to open an issue or reach out to the maintainers!
